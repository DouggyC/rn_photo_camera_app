# React Native Photo Camera App

Tech Stack: 
> React Native, expo


### Description
<hr>

## Photo App for Android and iOS. 

User can take and save photos to local device folder. 
User can access local folder and view photo details.

Required Permissions: 
- [x] Camera
- [x] Storage

<hr />
